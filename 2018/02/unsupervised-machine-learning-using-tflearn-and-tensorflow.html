Unsupervised machine learning using tflearn and tensorflow

Wesley Klock | Sun 25 February 2018

On the other hand, unsupervised learning has a more complex method to calculating and developing a model. It took me many tries to get the parameters with the best results and even then I still have a long way to go.

I simply didn't have enough computing power to run the tflearn model so I had to set up a ec2 AWS Ubuntu instance to run it on.

I used Bitfusion Ubuntu 14 TensorFlow system on a free-trial :) so it didn't cost me any money. Again, the code is quite complex but hopefully I can summarize the process with a few screenshots and my code.

Setting up my AWS system

One of my first tries on the AWS system using putty

My best try resulted in a binary accuracy of .7344!

The settings I used on my best try are recorded below in the code.

This model used tflearn's Deep Neural Net, I experimented with different numbers of layers, types of layers, etc. I settled on a network with 3 fully connected nets, one for reshaping, and lastly a regression layer.

The commented code is what I started with.

The last part of the code called Testing Phase takes in a tweet's relevant data and predicts a difference value
